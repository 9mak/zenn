## 🎧 内容

### AIに何を求めているか

| ビジョン | 日本語 | 割合 |
| ---- | ---- | ---- |
| Professional excellence | 仕事でより良い成果を出したい | 18.8% |
| Personal transformation | 自己変革・精神的な成長 | 13.7% |
| Life management | 生活管理・頭の余裕を作りたい | 13.5% |
| Time freedom | 家族や趣味のための時間を取り戻したい | 11.1% |
| Financial independence | 経済的な自由 | 9.7% |
| Societal transformation | 社会課題の解決 | 9.4% |
| Entrepreneurship | 起業・ビジネスの立ち上げ | 8.7% |
| Learning & growth | 学習・スキルアップ | 8.4% |
| Creative expression | 創作活動 | 5.6% |

### 心配していること

不安は複数回答で、一人あたり平均2.3個の懸念を話していたそうです。

| 懸念 | 日本語 | 割合 |
| ---- | ---- | ---- |
| Unreliability | 信頼性（ハルシネーションなど） | 26.7% |
| Jobs & economy | 雇用・経済への影響 | 22.3% |
| Autonomy & agency | 自律性・主体性の喪失 | 21.9% |
| Cognitive atrophy | 思考力・スキルの低下 | 16.3% |
| Governance | ガバナンスの不足 | 14.7% |

| Meaning & creativity | 人間の意味・創造性の喪失 | 11.7% |

### 希望と不安は同じ人の中に同居している

これが一番面白かったところで、楽観的な人と悲観的な人が別々にいるのではなく、**同じ人の中に希望と不安の両方がある**というのがデータで出ています。

たとえばAIの感情サポートに価値を感じている人は、AIへの依存を怖いと思っている割合が3倍高い。学習メリットを感じている人ほど思考力が落ちることも心配している、みたいなパターンが複数の軸で確認されていました。

### 地域によって期待が違う

途上国ではAIを「チャンスを作るもの」として見ている人が多く、先進国では「ごちゃごちゃした生活を整理するもの」として見ている傾向がありました。

東アジア（日本含む）は「自分を変えたい」という期待が全地域で一番高く（19%）、同時に思考力が落ちることへの不安も高かったです。AIへの向き合い方を個人レベルで強く意識しているのかもしれないと思いました。

## 💎 収穫

地域差のデータが個人的に一番面白かったです。途上国ほどEntrepreneurshipへの期待が高くて、先進国はLife managementが上位に来る傾向があって、AIに何を求めるかって生活環境でここまで変わるんですね。

Societal transformationが9.4%いたのも気になって、AIで病気を治したいとか教育格差をなくしたいとか、そういうスケールで考えている人がそこそこいるのは普通に面白いなと思いました。
